Política de Privacidad

Punto Radal Sync

Última actualización: 18 de marzo de 2026

1. Información que recopilamos

Punto Radal Sync accede a los siguientes datos de su cuenta de Clover:

  • Información de pagos procesados (monto, método de pago, fecha, resultado)
  • Información de órdenes asociadas a pagos
  • Datos básicos del comercio (nombre, identificador)

2. Cómo usamos la información

Los datos se utilizan exclusivamente para:

  • Generar reportes internos de conciliación de ventas
  • Comparar pagos digitales con registros del punto de venta
  • Detectar discrepancias en el registro de ventas

3. Almacenamiento de datos

Todos los datos se almacenan en una base de datos PostgreSQL local, alojada en infraestructura propia del comercio. No se utilizan servidores en la nube de terceros para almacenar datos de Clover.

4. Compartición de datos

No compartimos datos con terceros. La información obtenida de Clover se usa únicamente para fines internos del negocio y no se vende, alquila ni transfiere a ninguna otra entidad.

5. Seguridad

Implementamos las siguientes medidas de seguridad:

  • Tokens de acceso OAuth 2.0 almacenados de forma segura en base de datos
  • Refresh automático de tokens (los tokens de acceso tienen expiración)
  • Acceso a la base de datos restringido por credenciales y red local

6. Retención de datos

Los datos de pagos se retienen de forma indefinida como parte de los registros contables del negocio, conforme a las obligaciones fiscales argentinas.

7. Sus derechos

Como comerciante propietario de los datos, usted puede en cualquier momento:

  • Solicitar la eliminación de datos sincronizados
  • Revocar el acceso de la aplicación desde el panel de Clover
  • Solicitar un export de los datos almacenados
